# Instagram Clone SQL Analytics Project

## Overview
Designed and analyzed an Instagram Clone relational database using MySQL.

## Project Files

- instagram_db_schema.sql → Database design
- instagram_data.sql → Sample data
- instagram_queries.sql → 25+ business SQL queries

## SQL Skills Used

- Joins
- Subqueries
- CTEs
- Window Functions
- Aggregate Functions
- Constraints
- Data Analysis

## Key Questions Solved

- Oldest users
- Inactive users
- Most liked photos
- Top hashtags
- Mutual followers
- Users with above-average engagement
